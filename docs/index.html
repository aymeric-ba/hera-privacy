Politique de confidentialité

Dernière mise à jour : 28 juin 2026

Hera est une application de culture générale. Cette page décrit comment l'application gère vos données.

Aucune collecte de données personnelles

Hera ne collecte, ne transmet et ne stocke aucune donnée personnelle sur des serveurs externes. L'application fonctionne entièrement hors ligne.

Données stockées localement sur votre appareil

L'application enregistre uniquement les informations suivantes sur votre appareil, sans les transmettre :

  • Vos domaines préférés (histoire, sciences, arts, mythologie)
  • Les articles que vous avez déjà lus
  • Votre progression et vos statistiques de quiz

Ces données restent sur votre iPhone et sont supprimées si vous désinstallez l'application.

Aucun service tiers

Hera n'intègre aucun outil d'analyse, de publicité ou de pistage. Aucun SDK tiers ne collecte des données vous concernant.

Enfants

L'application ne collecte aucune donnée, y compris pour les utilisateurs de moins de 13 ans.
